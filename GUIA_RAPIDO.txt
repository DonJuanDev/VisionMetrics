📱 MENU PRINCIPAL (7 seções principais):
═══════════════════════════════════════════════════════════

1. DASHBOARD
   - Métricas em tempo real
   - Gráficos de conversão
   - Atualiza automaticamente a cada 5s

2. LEADS
   - Visualização: Lista ou Kanban
   - Filtros: busca, etapa, tags
   - Exportar dados
   - Ver perfil 360° do lead

3. CONVERSAS
   - Histórico WhatsApp
   - Detecção automática de vendas
   - Timeline de mensagens

4. VENDAS
   - Relatório de vendas
   - ROI por campanha
   - Faturamento

5. AUTOMAÇÃO
   ├─ Workflows: Automações baseadas em gatilhos
   └─ Tarefas: Gestão de tarefas por lead

6. ANALYTICS
   ├─ Atribuição Multi-Touch: 6 modelos diferentes
   ├─ Funil de Conversão: Taxas por etapa
   └─ Coortes: Análise temporal

7. CONFIGURAÇÕES
   ├─ Geral: Workspace
   ├─ Billing: Planos e assinaturas
   ├─ Membros: Usuários do workspace
   ├─ Integrações: Meta Ads, GA4
   ├─ WhatsApp: Números conectados
   ├─ Tags: Etiquetas coloridas
   ├─ Campos: Campos customizados
   ├─ Links & QR: Rastreamento
   ├─ API Keys: Acesso programático
   └─ LGPD: Compliance e privacidade

🔧 CONFIGURAÇÃO INICIAL:
═══════════════════════════════════════════════════════════

1. Configurações > Billing: Escolher plano
2. Configurações > Integrações: Conectar Meta Ads / GA4
3. Configurações > WhatsApp: Conectar número
4. Configurações > Tags: Criar tags
5. Configurações > API Keys: Gerar chave
6. Adicionar script no site com sua API key
